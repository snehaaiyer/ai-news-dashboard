# 🧠 Daily AI News Wrap Up

## 🚀 Overview

**Daily AI News Wrap Up** is a lightweight dashboard that curates the most important updates in Artificial Intelligence — covering both **Global** and **India-specific developments**.

The goal:
👉 Cut through noise
👉 Surface high-signal updates
👉 Make AI news consumable in minutes

---

## 🌍 What You Get

### 🔹 Global AI Updates

* Major announcements from companies like OpenAI, Google, Meta
* Breakthroughs in LLMs, robotics, and research
* Funding, acquisitions, and global policy moves

### 🇮🇳 India AI Updates

* Government initiatives & policy changes
* Indian startups & funding news
* Enterprise AI adoption across sectors

---

## ⚡ Key Features

* 📰 Curated daily news feed (no fluff)
* 🌎 Segmented view: Global vs India
* ⏱️ Quick-scan format (built for speed)
* 📊 Clean, minimal UI using Streamlit
* 🔄 Regular updates

---

## 🛠️ Built With

* 🐍 Python
* 📊 Streamlit
* 🌐 News APIs / Web scraping
* 🧠 Basic NLP for filtering & summarization

---

## 🧩 Use Cases

* Product Managers tracking AI trends
* Founders & Operators
* Students & Researchers
* Anyone who wants **signal > noise**

---

## ▶️ Run Locally

```bash
git clone https://github.com/snehaaiyer/ai-news-dashboard.git
cd ai-news-dashboard
pip install -r requirements.txt
streamlit run app.py
```
